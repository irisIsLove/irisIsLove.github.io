3-队列

基本概念

定义

一种操作首先的线性表，只允许在表的一端进行插入，而在表的另一端进行删除

基本操作

InitQueue(*Q) 初始化队列：构造一个空队列
DestoryQueue(*Q) 销毁队列：销毁并释放队列 Q 所占用的内存空间
EnQueue(*Q, x) 入队：若队列 Q 未满，x 加入成新的队尾
DeQueue(*Q, *x) 出队：若队列 Q 未空，删除当前队头，x 返回
GetHead(*Q, *x) 读队头：x 返回队头元素
QueueEmpty(*Q) 判断队列：判断 Q 是否为空，返回 bool 值

循环队列

  • 定义：将顺序队列臆造成为一个环状空间，把存储队列元素的表从逻辑上堪称一个环。当队首指针 Q.front = MaxSize - 1 后，再前进一个位置就自动到 0
  • 特点：克服了“假上溢”现象
  • 上溢：队列空间已满，而继续往后队列中插入元素，使数组越界而导致程序代码被破坏
  • 假上溢：由于队尾指针不断前移，超出向量空间。这时整个向量空间及队列使空的却产生“上溢”现象
  • 入队：rear = (rear + 1) MOD MaxSize
  • 出队：front = (front + 1) MOD MaxSize

区分队空、队满的方法

  • case1：牺牲一个单元（上述代码中的方法）
    • 队空：Q.rear == Q.front
    • 队满：(Q.rear + 1) % MaxSize == Q.front
    • 队列个数：N = (Q.rear + MaxSize - Q.front) % MaxSize
  • *case2：引入 size
    • 在 SqQueue 的结构体中引入“size”字段，表明队列的数据成员数量
    • 队空：Q.size == 0
    • 队满：Q.size == MaxSize、Q.rear == Q.front
  • *case3：引入 tag
    • 在 SqQueue 的结构体中引入“tag”字段，删除操作令 tag 为 0，增加令 tag 为 1
    • 队空：front == rear && tag == 0
    • 队满：front == rear && tag == 1

双端队列

  • 定义：允许从两端插入、两端删除的线性表
  • *分类：
    • 正常的双端队列
    • 输入受限的双端队列
    • 输出受限的双端队列

队列的应用

  • 队列在层次遍历中的应用 → 使用队列使为了保存下一步的处理顺序
  • 队列在操作系统中的应用 → 数据缓冲区
  • *队列在排序算法的应用 → 基数排序

栈和队列的共同点

逻辑结构相同，都可以采用顺序存储或链式存储，均只匀速在端点处插入和删除元素
