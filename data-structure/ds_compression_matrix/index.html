3-特殊矩阵的压缩存储

数组

  • 定义：由 n(n ≥ 1)个相同类型的数据元素构成的有限序列

  • 一个数据的所有元素在内存中还在那用一段连续的存储空间

  • 地址计算：

    • 设三位数组 demo[A][B][C]，且 A[0][0][0]的地址为 LOC，则元素数据 demo[i][j][k]的存储地址 A 为：
    • 行优先：$A = LOC + (i \times B \times C + j \times C + k)$
    • 列优先：$A = LOC + (k \times B \times A + j \times A + i)$

  • 普通矩阵存储：直接采用二维数据即可

对称矩阵(注：采取下三角存取)

  • 特性：$a_{ij} = a_{ji}$
  • 思想：将一个三角区存入一维数组
  • 元素个数：$\frac{n(n-1)}{2}$
  • 映射：
    $
    \begin{cases}
    \frac{i(i-1)}{2} + i - 1, i \geq j \
    \frac{j(j-1)}{2} + i - 1, i < j
    \end{cases}
    $
