2-链表

单链表

定义

对每个链表节点，除了存放元素自身的信息之外，还需要存放一个指向其后继的指针

头节点：

  • 含头：

    1. 为了操作上的方便，在单链表第一个节点之前附加一个几点；
    2. 数据域可以不舍任何信息，但可以记录表长等信息；
    3. 声明时头节点只想的第一个节点为空，下一个节点才开始存放数据。

  • 不含：
    写代码麻烦。头节点只想的第一个节点就开始存放数据

循环链表

定义

循环单链表和单链表的区别在于，表中最后一个节点的指针不是 NULL，而改为指向头节点，从而整个链表形成一个环

特点

  1. 判空条件不是头节点的指针是否为空，而是它是否等于头指针 
    •  bool isTail(LinkList L, LNode* p) {
   if (p->next == L)
     return true;
   else
     return false;
 }

*静态链表

定义

分配一整片连续的内存空间，各个节点集中安置

组成

数据元素 + 游标；游标充当指针作用，游标是 x，则该节点的后继的位置为 x

地址

首节点存放地址 + 空间 * 游标

特点

优：增、删、操作不用移动大量元素
缺：不能随机存取，只能从头结点以此往后插找；容量固定不变

应用

不支持指针的低级语言；数据元素数量固定不变的场景
