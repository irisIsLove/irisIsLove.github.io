gcc 常用参数

  • -v 查看 gcc 版本号， –version 也可以
  • -E 生成预处理文件
  • -S 生成汇编文件
  • -c 只编译，生成 .o 文件，通常称为目标文件
  • -I 指定头文件所在的路径
  • -L 指定库文件所在的路径
  • -l 指定库的名字
  • -o 指定生成的目标文件的名字
  • -g 包含调试信息，使用 gdb 调试需要添加 -g 参数
  • -On n=0~3 编译优化，n 越大优化程度越高，但编译时间越长
